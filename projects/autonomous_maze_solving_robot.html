Autonomous Maze-Solving Robot

Context: Academic Embedded Systems Project

What it is

An autonomous mobile robot designed to navigate and solve a maze using onboard sensors and embedded control logic. The system operates without external control, making decisions in real time based on environmental feedback.

Problem

  • Navigate an unknown maze without prior map information.
  • Reliably detect walls and openings using noisy sensor data.
  • Maintain stable motion and turning accuracy while making decisions.

My contributions

  • Implemented embedded firmware to read and filter sensor data in real time.
  • Designed control logic to handle wall detection, turning decisions, and dead-end recovery.
  • Integrated motor drivers and feedback to achieve consistent movement and turns.
  • Debugged timing, sensor noise, and edge cases during live maze runs.

Technical details

  • Controller: Microcontroller managing sensors, motors, and decision logic.
  • Sensing: Distance or line sensors used for wall detection and path decisions.
  • Control: Rule-based or algorithmic maze-solving strategy (e.g., left-hand rule).
  • Motor control: PWM-based speed control and direction logic.

Results

  • Successfully navigated and solved the maze autonomously.
  • Demonstrated reliable operation across multiple runs.
  • Improved performance through iterative tuning and debugging.
